The CSS Position Property

There are five different position values: static relative fixed absolute sticky HTML elements are positioned static by default Relative is positioned relative to its normal position. Fixed is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled. Position absolute is relative to the nearest positioned ancestor. Sticky is position based on the user’s scroll position.
